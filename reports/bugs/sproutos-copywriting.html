Multiple UI feedback strings are hardcoded in inline JavaScript. They are never passed through wp_json_encode( __() ) and are invisible to any translation tool. Any site on a non-English locale will always see raw English text in these feedback states.

LineHardcoded String
1027'Error'
1029'Network error. Please try again.'

2042'Error copying'
3210'Loading...'
4297'Failed to load details.'
4305'Failed to purge logs.'
4306'Logs purged.'
4787'Loading...'

5097'Error'
Fix Pattern — apply to every string above
